Definition
It's important to provide useful and descriptive labels that explain the meaning and purpose of each interactive element to users. When labeling graphical elements, such as ImageView and ImageButton objects, use the android:contentDescription XML attribute for static elements and the setContentDescription() method for dynamic elements. These labels allow screen readers, such as TalkBack, to properly explain the function of a particular control to users who rely on these services. More info...
